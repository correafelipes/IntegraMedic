# IntegraMedic - Software de Gestão de Clínicas Médicas

**Descrição:** 
O IntegraMedic é um software SaaS desenvolvido para auxiliar a gestão de clínicas e consultórios médicos (modelo clone Feegow). Ele visa unificar em uma única plataforma a recepção (cadastro, agendamentos, faturamento) e o corpo clínico (prontuário eletrônico completo).

## Stack (Previsão)

- **Backend:** Node.js (Express).
- **Banco de Dados:** PostgreSQL.
- **Ambiente:** Docker & Docker Compose (para padronização entre Dev e Produção).
